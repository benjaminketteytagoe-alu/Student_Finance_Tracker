NauriTrack

NauriTrack is an intuitive and comprehensive student finance management tool built with vanilla HTML, CSS, and JavaScript. It empowers students to take full control of their finances by allowing them to track daily expenses, set and monitor budgets, and analyze spending trends through clear visual With its user-friendly interface and interactive dashboard, NauriTrack helps users identify where their money goes, make informed financial decisions, and build better saving habits. NauriTrack Key Features

  • Real-time dashboard with spending analytics
  • Budget tracking with live alerts
  • Advanced regex-powered search

  • Multi-currency support
  • Fully accessible (WCAG compliant)
  • Responsive mobile-first design

Regex Examples

  • /coffee|tea/i - Find beverages
  • /\.\d{2}/ - Find amounts with cents
  • /\b(\w+)\s+\1\b/ - Find duplicate words
  • /^[A-Z]/ - Find capitalized entries
